## Correr Migraciones de base de datos
```bash
#Generar
$ npm run migration:generate

#Correr
$ npm run migration:run
```

## Running the app

```bash
#Construir
$ npm run build

# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
